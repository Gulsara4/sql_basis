**Day 00:** Изучалась реляционная модель данных и базовые конструкций SQL для выборки информации из связанных таблиц.
**Day 01:** Изучалась фильтрация данных с помощью подзапросов и операторов NOT IN и NOT EXISTS.

**Day 02:** Изучались сложные JOIN и многоуровневые подзапросы для фильтрации данных по нескольким условиям.

**Day 03:** Изучалось изменение данных с помощью DML команд и работа с общими табличными выражениями (CTE).

**Day 04:** Изучалось создание виртуальных представлений (VIEW) и материализованных представлений (MATERIALIZED VIEW).

**Day 05:** Изучалось создание и применение частичных индексов для оптимизации производительности запросов.
**Day 06:** Изучалось добавление бизнес-правил через ограничения таблиц (constraints) и значения по умолчанию.

**Day 07:** Изучалась OLAP-аналитика: группировки, агрегатные функции и вычисление метрик.

**Day 08:** Изучались транзакции и уровни изоляции (repeatable read) для согласованности данных.

**Day 09:** Изучалось создание пользовательских функций и работа с массивами в PL/pgSQL.

**Day t0 (командный проект):** Изучалось рекурсивное решение задачи коммивояжёра для поиска оптимального маршрута.

**Day t1 (командный проект):** Изучалось создание ETL-процесса и обработка временных рядов в DWH.
